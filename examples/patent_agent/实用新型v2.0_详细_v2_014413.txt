专利申请文件

发明名称：一种多功能折叠椅

技术领域：本实用新型涉及家具领域，具体涉及一种具有折叠、调节和储物功能的多功能折叠椅。

背景技术：
目前市场上的折叠椅种类繁多，但大多功能单一，结构设计不够合理。常见的折叠椅通常只具备简单的折叠功能，占用空间仍然较大。一些具有调节功能的椅子，其调节方式较为单一，舒适度不高。同时，多数折叠椅缺乏储物功能，无法满足人们存放小物件的需求。因此，现有折叠椅在结构上存在以下问题：
1. 折叠方式不够优化，折叠后体积仍然较大，不便于携带和存放。
2. 靠背调节功能不足，无法满足不同用户的舒适度需求。
3. 缺乏储物空间，无法满足用户存放随身物品的需求。
4. 结构强度不足，长期使用容易出现松动和损坏。
这些结构上的不足限制了现有折叠椅的使用场景和用户体验，因此，需要一种结构更加紧凑、功能更加丰富、使用更加舒适的多功能折叠椅。

实用新型内容：
要解决的技术问题：
本实用新型旨在克服现有技术中的不足，提供一种结构更加紧凑、功能更加丰富、使用更加舒适的多功能折叠椅。具体要解决的技术问题包括：
1. 如何优化折叠结构，减小折叠后的体积，提高便携性。
2. 如何实现靠背的多档位调节，满足不同用户的舒适度需求。
3. 如何增加储物空间，方便用户存放随身物品。
4. 如何提高整体结构的强度和稳定性，延长使用寿命。
针对现有折叠椅的结构缺陷，本实用新型重点解决以下结构问题：
*   现有折叠椅的折叠方式较为简单，通常只是将椅面和靠背简单折叠，折叠后的体积较大，占用空间多，不便于携带和存放。**本实用新型要解决的问题是：如何设计一种更加紧凑的折叠机构，使折叠后的体积最小化。**
*   现有折叠椅的靠背调节功能通常比较单一，只能进行小范围的角度调节，无法满足不同用户的需求。**本实用新型要解决的问题是：如何设计一种多档位可调节的靠背机构，使用户可以根据自身需求调节靠背角度，获得最佳的舒适度。**
*   现有折叠椅通常不具备储物功能，用户无法方便地存放随身物品。**本实用新型要解决的问题是：如何在折叠椅的结构中巧妙地设计储物空间，方便用户存放书籍、遥控器等小件物品。**
*   现有折叠椅的结构强度和稳定性有待提高，长期使用容易出现松动和损坏，影响使用寿命。**本实用新型要解决的问题是：如何采用高强度材料和合理的结构设计，提高折叠椅的整体强度和稳定性。**

技术方案：
本实用新型采用以下技术方案实现：
1. 椅座采用双层折叠结构：椅座由上下两层组成，通过铰链连接，可向上翻折，减小折叠后的体积。
2. 靠背采用五档角度调节机构：靠背通过可调节角度的连接件与椅座连接，实现90度至135度范围内的五档角度调节。连接件包括固定齿轮、可滑动齿轮和锁定机构，通过滑动可滑动齿轮与固定齿轮的不同位置啮合，实现角度调节，锁定机构用于固定选择的角度。
3. 扶手采用可旋转收纳设计：扶手通过旋转机构与椅座连接，平时支撑手臂，不用时可向下翻转收纳，节省空间。旋转机构包括旋转轴、固定套和锁定按钮，通过按下锁定按钮，可以解锁旋转轴，实现扶手的旋转。
4. 底座设置储物抽屉：底座内部设置一个抽屉，用于存放书籍、遥控器等小件物品。抽屉采用滑轨结构，方便抽拉。
5. 整体框架采用高强度铝合金材料：保证结构的稳固性和轻便性。铝合金框架采用焊接和螺栓连接，保证连接的可靠性。

有益效果：
本实用新型具有以下有益效果：
1. 折叠后体积小，便于携带和存放。双层折叠椅座和可收纳扶手设计，大大减小了折叠后的体积，方便用户在不同场景中使用和存放。
2. 靠背角度可调节，满足不同用户的舒适度需求。五档角度调节机构使用户可以根据自身的需求调整靠背角度，获得最佳的坐姿体验，减轻长时间坐姿带来的疲劳感。
3. 具有储物功能，方便用户存放随身物品。底座抽屉设计方便用户存放书籍、遥控器等小件物品，提高了使用的便利性。
4. 结构稳固，使用寿命长。高强度铝合金框架保证了整体结构的稳固性和耐用性，延长了产品的使用寿命。
5. 操作简单，使用方便。各个功能模块的设计都充分考虑了用户的操作习惯，使得用户可以轻松地进行折叠、调节和储物等操作。

具体实施方式：
下面结合附图和实施例对本实用新型作进一步详细说明。
如图1所示，本实用新型一种多功能折叠椅，包括椅座1、靠背2、扶手3和底座4。
椅座1：椅座1由上层椅座11和下层椅座12组成，上层椅座11和下层椅座12之间通过铰链13连接，实现向上翻折的功能。在上层椅座11的下方设置有加强筋，提高椅座的承重能力。
靠背2：靠背2通过连接机构与椅座1连接。如图3所示，连接机构包括固定齿轮21、可滑动齿轮22和锁定机构23。固定齿轮21固定在椅座1的后侧，可滑动齿轮22与靠背2连接，并且可以在固定齿轮21上滑动。锁定机构23用于固定可滑动齿轮22的位置，实现靠背角度的锁定。用户可以通过滑动可滑动齿轮22，使其与固定齿轮21的不同位置啮合，实现靠背角度的调节。
扶手3：扶手3通过旋转机构与椅座1连接。如图4所示，旋转机构包括旋转轴31、固定套32和锁定按钮33。旋转轴31与扶手3连接，固定套32固定在椅座1的侧面。旋转轴31可以在固定套32内旋转，实现扶手的收纳功能。锁定按钮33用于锁定旋转轴31的位置，防止扶手在使用过程中意外旋转。
底座4：底座4内部设置一个抽屉41，用于存放书籍、遥控器等小件物品。如图5所示，抽屉41通过滑轨42与底座4连接，方便抽拉。
整体框架：整体框架采用高强度铝合金材料，通过焊接和螺栓连接，保证结构的稳固性和轻便性。

使用时，展开椅座1，将扶手3旋转至支撑位置，调节靠背2的角度至舒适位置，即可正常使用。需要收纳时，按下锁定按钮33，将扶手3向下旋转收纳，滑动可滑动齿轮22，将靠背2放倒，向上翻折椅座1，即可完成折叠。
